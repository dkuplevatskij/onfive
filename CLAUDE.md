# OnFive — AI-репетитор «на пятёрку»

## Бренд
- **Название:** OnFive
- **Слоган:** «Учись на пятёрку с AI»
- **Философия:** Сократический метод — никогда не давать готовых ответов. Вести ученика подсказками и наводящими вопросами к самостоятельному решению.
- **Целевая аудитория:** ученики 5–11 классов российских школ + их родители
- **Дизайн:** Apple Cupertino — белый фон (#F2F2F7), системные шрифты, blur-эффекты, iOS-стилистика
- **Домены:** onfive.ru (основной), onfive.ai (tech), onfive.app (мобильный)

---

## Стек

### Frontend
- React 18+ с TypeScript
- Vite (сборка)
- React Router (навигация)
- Tailwind CSS (стилизация в Apple-стиле)
- KaTeX (рендеринг формул)
- marked (рендеринг Markdown)
- @telegram-apps/sdk (Telegram Mini App)
- Web Speech API (голосовой ввод)
- SpeechSynthesis API (озвучка ответов)

### Backend
- Node.js + Fastify (или Express) + TypeScript
- Anthropic Claude API (Sonnet для чата, Haiku для лёгких задач)
- OpenAI Whisper API (транскрипция голоса — fallback для Web Speech API)
- ElevenLabs или Cloud TTS (озвучка — опционально)

### База данных
- PostgreSQL через Supabase
- Таблицы: users, sessions, messages, achievements, parent_reports
- Supabase Auth (включая Telegram Login)
- Supabase Realtime (опционально)

### Деплой
- Frontend: Vercel или Cloudflare Pages
- Backend: Railway, Fly.io или Render
- CI/CD: GitHub Actions

## Функционал

### 1. Выбор класса и предмета
- При первом входе ученик выбирает класс (5–11)
- Класс можно изменить в настройках
- Предметы зависят от класса:
  - **5–6 класс:** Русский язык, Литература, Математика, История, Биология, География, Английский, Информатика, Обществознание
  - **7 класс:** + Алгебра, Геометрия, Физика (вместо Математики)
  - **8 класс:** + Химия
  - **9 класс:** все предметы + подготовка к ОГЭ
  - **10–11 класс:** все предметы + подготовка к ЕГЭ (профильные)

### 2. Навигация: Предмет → Тема → Режим → Чат
- Каждый предмет содержит темы/параграфы по ФГОС
- Темы организованы по разделам (главам учебника)
- Есть кнопка «Своя тема» для ввода произвольной темы

### 3. Режимы обучения (5 штук)
1. **📖 Объяснить тему** — пошаговое объяснение с проверочными вопросами
2. **✏️ Помощь с домашкой** — подсказки и наводящие вопросы, НЕ готовые ответы
3. **✅ Проверь меня** — вопросы по одному, разбор ошибок
4. **🎯 К контрольной/экзамену** — план повторения, тренировочные задания
5. **💬 Свободный вопрос** — любой вопрос по предмету

### 4. Генерация докладов (НОВОЕ)
- Отдельный раздел «Доклады» в навигации
- Ученик указывает: предмет, тему доклада, объём (краткий / средний / подробный)
- AI генерирует **структуру** доклада (план), а не готовый текст
- Затем ученик может раскрыть каждый пункт плана — AI помогает сформулировать мысли
- Режимы доклада:
  - **Помоги написать** — AI задаёт вопросы, ученик отвечает, AI помогает оформить ответы в связный текст
  - **Сгенерируй черновик** — AI создаёт черновик, ученик редактирует (для старших классов)
  - **Проверь мой доклад** — ученик вставляет свой текст, AI даёт обратную связь
- Финальный доклад можно скачать как .docx или .pdf
- В докладе автоматически указывается: «Подготовлено с помощью OnFive AI»

### 5. Голосовой ввод и озвучка
- Кнопка микрофона рядом с полем ввода
- Web Speech API для распознавания русской речи
- Fallback на Whisper API (через бэкенд) при плохом качестве
- Озвучка ответов AI через SpeechSynthesis API
- Кнопка 🔊 на каждом сообщении AI для прослушивания

### 6. Геймификация
- **XP:** +5 за сообщение, +10 бонус за сессию (5+ сообщений), +15 за правильный ответ в режиме «Проверь меня»
- **Уровни:** Новичок (0) → Ученик (100) → Знаток (300) → Мастер (700) → Профессор (1200) → Гений (2000)
- **Серия (streak):** дни подряд, огонёк 🔥
- **Достижения:** Первый шаг, Неделя знаний, Упорство (30 дней), Полиглот (3+ предмета), Эрудит (все предметы), Марафонец (50 занятий), Сотня (100 сообщений), Полтыщи (500 XP), Автор (первый доклад), Докладчик (10 докладов)

### 7. Родительская панель
- Защищена 4-значным PIN-кодом
- Статистика: серия дней, XP, количество занятий, охват предметов
- График активности по предметам
- Лог последних занятий (дата, предмет, тема, режим, длительность)
- Список сгенерированных докладов
- Блок «Методика» — объяснение сократического подхода

### 8. Подготовка к ОГЭ/ЕГЭ (9, 10–11 классы)
- Специальный раздел для каждого предмета
- Формат заданий ОГЭ/ЕГЭ (тестовая часть, развёрнутые ответы)
- Таймер на выполнение
- Разбор типичных ошибок

## Системные промпты

### Базовый промпт (вставляется всегда)
```
Ты — дружелюбный репетитор для ученика {grade} класса российской школы.
Предмет: «{subject}». Тема: «{topic}».

ГЛАВНОЕ ПРАВИЛО: НИКОГДА НЕ ДАВАЙ ГОТОВЫХ ОТВЕТОВ И РЕШЕНИЙ.
- Веди ученика к ответу через наводящие вопросы и подсказки.
- Полное решение — ТОЛЬКО после 2-3 попыток ученика И его явной просьбы.

Правила:
- Уровень {grade} класса, программа ФГОС.
- Примеры из жизни, понятные подростку.
- Мотивируй, хвали за правильные шаги. Обращайся на «ты».
- Формулы: $...$ строчные, $$...$$ блочные (LaTeX).
- «Не понимаю» → объясни ИНАЧЕ, другим способом.
- Только по предмету.
```

### Промпт для режима «Доклады — Помоги написать»
```
Режим: ПОМОЩЬ С ДОКЛАДОМ.
- Сначала уточни тему и объём.
- Предложи план доклада (введение, 3-5 пунктов, заключение).
- По каждому пункту задавай вопросы ученику: «Что ты знаешь о...?», «Как ты думаешь, почему...?»
- На основе ответов ученика помоги сформулировать абзац.
- НЕ пиши весь доклад сам. Ученик должен участвовать в создании каждой части.
- Результат: текст, который ученик понимает, потому что он сам помогал его создавать.
```

### Промпт для режима «Доклады — Черновик»
```
Режим: ГЕНЕРАЦИЯ ЧЕРНОВИКА ДОКЛАДА.
- Создай структурированный доклад по теме.
- Объём: {length} (краткий ~500 слов / средний ~1000 слов / подробный ~2000 слов).
- Структура: Введение → Основная часть (3-5 разделов) → Заключение → Список источников.
- Язык: грамотный, но на уровне {grade} класса, не академический.
- Используй факты и примеры.
- В конце предложи ученику прочитать и задать вопросы, если что-то непонятно.
```

- Интеграция через @telegram-apps/sdk
- Авторизация через Telegram Login (initData)

## Фазы разработки

### Фаза 1 — MVP (5-6 недель)
- Инициализация монорепо, настройка сборки
- React-приложение с роутингом
- Выбор класса (5-11) и предмета
- Темы по ФГОС для всех классов
- Чат с Claude API (5 режимов)
- Базовая геймификация (XP, уровни, streak)
- Деплой как веб-приложение

### Фаза 2 — Доклады + голос (3-4 недели)
- Генерация докладов (3 режима)
- Экспорт в .docx и .pdf
- Голосовой ввод (Web Speech API)
- Озвучка ответов
- Родительская панель с PIN

### Фаза 3 — Telegram + App Store (4-6 недель)
- Telegram Mini App интеграция
- Capacitor обёртка для iOS
- Push-уведомления (напоминания об учёбе)
- Подписочная монетизация

### Фаза 4 — ОГЭ/ЕГЭ + масштабирование
- Тренажёр ОГЭ/ЕГЭ с таймером
- OCR: фото задачи → распознавание → помощь
- Мультитенантность для школ
- Аналитика для учителей

## Начало работы

```bash
# Инициализация
mkdir onfive && cd onfive
npm init -y

# Создание воркспейсов
mkdir client server shared

# Клиент
cd client
npm create vite@latest . -- --template react-ts
npm install react-router-dom zustand katex marked
npm install -D tailwindcss @tailwindcss/vite

# Сервер
cd ../server
npm init -y
npm install fastify @anthropic-ai/sdk @supabase/supabase-js dotenv
npm install -D typescript @types/node tsx
```
